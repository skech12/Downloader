# Contributing to Neurvance Downloader

Hey — glad you're here. There are two main ways to contribute:

1. **Request a dataset** you want to see added
2. **Report a bug** with the downloader itself

---

## Requesting a Dataset

The fastest way is a GitHub Issue.

**→ [Open a dataset request](https://github.com/skech12/Downloader/issues/new?template=dataset_request.md)**

What to include:
- What the dataset is (topic, type of data — Q&A, conversations, instructions, etc.)
- Why it's useful for fine-tuning
- Any source you know of (we'll handle cleaning and formatting)
- Approximate size if known

We review requests weekly. If it's a good fit and CC0-compatible, it gets added to the pipeline.

---

## Discussing Dataset Ideas

Not sure if your idea fits? Start a [GitHub Discussion](https://github.com/skech12/Downloader/discussions) — the **Dataset Requests** category is the right place.

Great for:
- Brainstorming dataset ideas before filing an issue
- Asking about existing datasets
- General Q&A about the API or fine-tuning workflows

---

## Reporting Bugs

Open a [GitHub Issue](https://github.com/skech12/Downloader/issues/new) with:
- What you ran
- What you expected
- What actually happened (error message, traceback, etc.)
- Python version + OS

---

## What We Don't Accept (Yet)

- Pull requests for new datasets directly — the pipeline is internal for now
- PRs that modify licensing or legal files

Code PRs for the downloader client itself are welcome though. Just open an issue first so we're aligned.

---

## Code of Conduct

Be decent. That's it.
